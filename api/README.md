# BlackRoad espocrm Enterprise API

## Overview

Proprietary API layer for BlackRoad espocrm Enterprise.

## Features

- **Authentication**: JWT, OAuth 2.0, SSO
- **Rate Limiting**: Tier-based limits
- **Analytics**: Usage tracking and billing
- **Webhooks**: Event-driven integrations

## Endpoints

```
POST /api/auth/login
GET  /api/espocrm/*
POST /api/espocrm/*
```

## Integration

Connects to upstream espocrm via standard APIs.
